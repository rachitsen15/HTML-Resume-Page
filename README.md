# HTML Resume Website

## Overview
This project is a single-page resume website built using only HTML.  
The focus is on clean structure, proper HTML semantics, and readability without using CSS or JavaScript.

## Technologies Used
- HTML5

## How to Run
1. Clone the repository  
2. Open `index.html` in any web browser  

No setup or installation is required.

## Features
- Single-page resume layout  
- Proper use of HTML semantic elements  
- Uses `<strong>`, `<em>`, `<b>`, and `<i>` tags for emphasis  
- Clean and readable structure
